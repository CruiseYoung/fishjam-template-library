块级元素block level element(如 h1,p,ul,table 等)， 浏览器显示时，通常会以新行来开始和结束
内联元素inline element(如 b,td,a,img 等)， 浏览器显示时，通常不会以新行开始

常见元素

格式 描述
a 超链接(anchor), "URL#锚名" 的方式可以直接跳转到指定 URL 的指定书签位置，通过 title 属性指定Tooltip
<href="http://www.baidu.com">指向另一个文档
<id="pageBottom"> 创建文档内的书签(也称为锚, 不可见,只用于定位)，然后通过 pageBottom链接跳转
MailSample: EMail Send "Hello World"
address
地址信息 1
地址信息 2
城市
blockquoteText quoted from some source
body内容区域
br/换行
div[块级]通用容器，文档中的节或区域(块级)，可以组合其他HTML元素，常用于 文档布局(如通过 float:left 等指定布局位置)，以便分别对这些区域进行格式化或精确定位
form表单
fieldset + legend带标题的框,类似GroupBox,甚至可以增加子表单
frameset + frame

框架，可以在同一个浏览器窗口中显示不止一个页面(垂直:cols; 水平:rows)

坏处：很难打印整张页面

现代Web开发中，不鼓励使用框架页，因为在不同的窗口尺寸时，框架页面可能无法很好地工作，且框架页不能被搜索引擎正确地查找。
h1~h6标题，爬虫机器人可以根据这些生成文档大纲(类似Word中的 文档结构图)
hr/水平线，可以利用 size 属性指定宽度
iframe 内联框架(HTML 页中的框架)，可通过 src 属性指向其他的 html 等
  • 设置name属性后，可作为链接的target
  • 有些老的浏览器不支持iframe
  • 可通过height和width设置高宽
  • frameborder=0 可移除边框
  • scrolling=yes|no|auto, 控制是否显示滚动条
img 图像高级用法(图像映射image-map -- 带有可供点击区域的图像地图。其中的每个星球都是一个超级链接)， 如将 href 设置为 javascript:void(0) 忽略鼠标点击，则可通过在onMouseOver中实现鼠标移动时的动态效果，
label 如果在label元素内点击文本,则会触发对应的控件( for属性=控件的id )
list(ul/ol/dl) 注意 li 的属性 float="none|left|right" 属性可以控制排列方式，可以做成标签页形式( 锋利的jQuery中 \jQuery_Code\05第五章\第五章例子\demo10-选项卡 )
  • 无序列表(ul -> li)

marquee可以显示移动内容(如滚动新闻等),direction=left|right,bihavior=scroll|slide|alternate, loo=n(循环次数)
p段落,浏览器会自动地在段落的前后添加空行
span[内联]表示嵌套在其他元素中的部分文本内容(如包含在标题元素或段落元素中的部分文本), 通常用来格式化所包含的文本片段
table表格, thead: tr(表格行) > th(表格头), tbody:tr(表格行)> td(表格数据), 属性中 colspan/rowspan 可跨行或跨列
