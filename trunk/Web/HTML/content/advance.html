<!DOCTYPE > 告知浏览器该文档使用哪种 HTML 或 XHTML 规范。该标签可声明三种 DTD 类型，分别表示不同的HTML 文档
  • 严格版本( strict.dtd | xhtml1-strict.dtd )
  • 过渡版本( loose.dtd | xhtml1-transitional.dtd )
  • 基于框架( frameset.dtd | xhtml1-frameset.dtd )
如 <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
声明了文档的根元素是 html，它在公共标识符被定义为 "-//W3C//DTD XHTML 1.0 Strict//EN" 的 DTD 中进行了定义，
浏览器将明白如何寻找匹配此公共标识符的 DTD。如果找不到，浏览器将使用公共标识符后面的 URL 作为寻找 DTD 的位置。
head

所有头部元素的容器，可包含脚本，样式表，元信息等等

<base target="_blank" />为页面上的所有链接规定默认地址(href)或默认目标(target),如本例在新窗口中打开,可参见 list.html
<link type=xxx href=xxx >定义文档与外部资源之间的关系，常用于连接CSS
<meta name=xxx content=xxx >提供关于 HTML 文档的元数据，提供 描述、关键词(Keywords)、作者(Author)、最后修改时间(revised) 等信息
<script >定义客户端脚本，如 JavaScript 等
<style >定义HTML 元素在浏览器中呈现的样式
meta中的refresh重定向到新的网址：如 <meta http-equiv="refresh" content="5;url=http://www.w3school.com.cn" > 将在5秒后重定向
object 或 embed插件,扩展浏览器标准功能，如 播放音乐、显示地图、验证银行账号，控制输入等
