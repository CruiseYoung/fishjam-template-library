010 Editor - Binary Template Archive

This page contains a repository of Binary Templates for use with 010 Editor. Please feel free to submit any templates you have that may be useful to other people. For more information on templates, see the Binary Templates page. Please review the terms and conditions for using this archive.


 Template File

Description










AVITemplate.bt
Parse an AVI movie file.







BMPTemplate.bt
Parse a Bitmap image.







CDATemplate.bt
Simple CDA file template to read Audio CD header information.







CLASSTemplate.bt
Parse Java Class (JVM) files.







CLASSTemplate2.bt
Alternate template for analyzing Java Class files (.class).







CRXTemplate.bt
Template for packaged google chrome extensions







DBFTemplate.bt
Parses .dbf (database) format files (used with ESRI shapes files)







DEXTemplate.bt
A template for analyzing Dalvik VM (Android) DEX files.







DMPTemplate.bt
Template to parse the header of a Microsoft memory dump file (DMP) as produced by the debugger and kernel crashdump facility.







EDIDTemplate.bt
Template for EDID files (Extended display identification data).







ELFTemplate.bt
Decode the ELF format for both 32/64 bit in big/little endian, decode the elf, program, and section headers. Also decode the synamic symbol table entries. Many fields implement custom viewers but do not handle writing from those views.







EMFTemplate.bt
Enhanced Metafile Format (EMF) template.







EVSBTemplate.bt
Decomposes Video Symbol Stream Files for E-8VSB modulation as per ATSC Digital Television Standard (A/53).







EVTTemplate.zip
Set of templates for parsing Microsoft Windows Event Log files (EVT).







EXETemplate.bt
Parse an executable file.







EXETemplate2.bt
An alternate template for EXE and DLL file headers. This template displays descriptive names for section flags and successfully reads Win64 executables. It also can open files compressed with FSG, recognize section names of the common compilers and packers, show overlay data, and find the 'Rich' signature of Microsoft linker.







exFATTemplate.bt
Microsoft exFAT file system Parser Template.







FAT16Template.bt
This template can find the partitions and display the FAT16 partitions on a disk.







FLVTemplate.bt
Template for Flash Video (FLV) files.







GIFTemplate.bt
Parse an image in GIF 89a Graphics Interchange Format.







GocleverTemplate.bt
Template for the GOCLEVER GPS Navigation log format.







ICOTemplate.bt
Template for ICO icon files.







InspectorDates.bt
Demonstrates how to add a series of dates to the Auto Inspector. Includes:
- WebkitTime - 64-bit int, microseconds since 01/01/1601
- HFSTime - 32-bit int, seconds since 01/01/1904
- AppleTime - 32-bit int, seconds since 01/01/2001
- PRTime - 64-bit int, microseconds since 01/01/1970
- JavaTime - 64-bit int, microseconds since 01/01/1970
- GPSTime - 32-bit int, seconds since 01/06/1980
- BlackberryDate - 32-bit int, minutes since 01/01/1900.







JPGTemplate.bt
Template for JPG image files.







LNKTemplate.bt
View data in a Microsoft shortcut (LNK) file.







LUKSTemplate.bt
Template for LUKS (Linux Unified Key Setup) hard disk encryption.







MBRTemplate.bt
Parse the Master Boot Record from a Harddisk.







MP3Template.bt
Parse an MP3 music file.







NetflowVersion5.bt
Parses Cisco's Netflow Version 5 format.







PALTemplate.bt
Template for Microsoft PAL palette files.







PCAPTemplate.bt
Parse a PCAP network capture file.







PCXTemplate.bt
Parse a PCX bitmap header.







PDFTemplate.bt
Template for Adobe PDF (Portable Document Format) files.







PNGTemplate.bt
Parse a PNG (Portable Network Graphics) file.







PNG12Template.bt
Parse all PNG 1.2 chunks.







PSFTemplate.bt
Parses a WSUS PSF (Patch Storage File) file.







PYCTemplate.bt
Parse python bytecode .pyc and .pyo files, support python 2.4 to 2.7.







RARTemplate.bt
Parses RAR 2.x, 3.x and SFX RAR archives.







RDBTemplate.bt
Defines a template for parsing QQ's RDB files.







SHPTemplate.bt
Parses ESRI shp files.







SHXTemplate.bt
Parses ESRI shx files.







SinclairMicrodriveImage.bt
Defines a template for parsing images of emulated Sinclair Microdrive cartridges.







SSPTemplate.bt
Define a template for parsing SmartSniff Packet files.







STLTemplate.bt
Parse an STL binary file containing 3D geometry (CAD).







SWFTemplate.bt
Parses Adobe Flash swf files.







TacxTemplate.bt
Template for all the Tacx Fortius cycling training software file formats, including pgmf, caf, imf, rlv and mrlv.







TOCTemplate.bt
Template to make sense of, and modifications to, Eudora 7.* (possible older versions too) mailbox index files (.toc).







TIFTemplate.bt
Template for TIFF (Tagged Image File Format) files.







TGATemplate.bt
Shows the fields of a TGA image.







WAVTemplate.bt
Parse a WAV sound file.







WAVTemplateAdv.bt
An advanced WAV template file that includes SMPL chunk parsing (Parsing wav files that contains Sustain Loops. Useful for Game Developers). Tested with: CoolEdit and Sound Forge.







WinhexPosTemplate.bt
Parse the winhex.pos file containing find results from the WinHex program.







WMFTemplate.bt
Parse the Windows Metafile (WMF) graphics file format.







ZIPTemplate.bt
Parse a Zip archive.







ZIPTemplateAdv.bt
Parses more complex Zip formats such as Zip64.
